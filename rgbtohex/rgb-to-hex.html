Description

RGB(red, green, blue) refers to a system for representing the colors to be used on a computer display. Red, green, and blue can be combined in various proportions to obtain any color in the visible spectrum. It's usually represented in decimal or hexadecimal number for each color.

Given a decimal for each color as input, we convert the given decimal into hex.

Some Examples:
Input: 255 255 255
Output: #FFFFFF

Input: 75 174 218
Output: #4BAEDA

Input: 0 21 69
Output: #001545
