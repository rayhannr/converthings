Binary to String

The input must contain 0 or 1 only

Also try : string to binary

Description

In mathematics, binary number has base 2 which means it is represented with 0-1 and here we are converting binary number into string to make easier for you to decode the binary-coded message sent to you.

Given a binary code as input, we convert the given binary into string.

Some Examples:
Input: 1001000 1100101 1111001 100001
Output: Hey!

Input: 1011001 1101111 1110101 100000 1101100 1101111 1101111 1101011 100000 1100111 1101111 1110010 1100111 1100101 1101111 1110101 1110011 101110
Output: You look gorgeous.

Input: 1010111 1101001 1101100 1101100 100000 1111001 1101111 1110101 100000 1101101 1100001 1110010 1110010 1111001 100000 1101101 1100101 111111
Output: Will you marry me?
