Binary to Hex

The input must contain 0 or 1 only.

Also try : hex to binary

Description

In mathematics, binary number has base 2 which means it is represented with 0 or 1 and hexadecimal or usually called with hex number has base 16, represented with 0-9 and A-F to represent 10-15.

Given a binary number as input, we convert the given binary into hexadecimal number.

Some Examples:
Input: 110
Output: 6

Input: 101010
Output: 2A

Input: 110010011101
Output: C9D
