Octal to Decimal

The input must contain number within 0-7 only.

Also try : decimal to any base

Description

In mathematics, octal number has base 8 which means it is represented with number within 0-7 and decimal number has base 10 and represented with numbers within range of 0-9 which is the most well-known number for being commonly used by people all around the world.

Given a octal number as input, we convert the given octal into decimal number.

Some Examples:
Input: 11
Output: 9

Input: 6235
Output: 3229

Input: 463142
Output: 157282
