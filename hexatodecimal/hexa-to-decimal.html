Hex to Decimal

The input must contain numbers or char within range of A-F only.

Also try : decimal to any base

Description

In mathematics, hex number has base 16 which means it is represented with numbers and A-F to represent 10-15 and decimal number has base 10 and represented with numbers within range of 0-9 which is the most well-known number for being commonly used by people all around the world.

Given a hex number as input, we convert the given hex into decimal number.

Some Examples:
Input: C9D
Output: 3229

Input: 4DA59
Output: 318041

Input: 921
Output: 2337
